🚀 Setup Admin User

Follow these steps to create your admin account in Firebase Firestore.

Your User UID:
6UEfM5O4WNgUBWKSw38hsdVtaah1

Step 1: Open Firebase Console

Click the button below to open your Firebase Firestore Database:

Open Firebase Firestore →

Step 2: Create User Document

  1. In Firestore, look for the "users" collection
    • If it exists, click on it
    • If not, click "Start collection" and name it users
  2. Click "Add document"
  3. In the "Document ID" field, paste this exactly:
    6UEfM5O4WNgUBWKSw38hsdVtaah1

Step 3: Add Fields

Add these 5 fields to the document:

Field Name Type Value
name string Admin User
email string Your login email
role string admin (lowercase!)
isBlocked boolean false
createdAt timestamp Click "Set to current time"
⚠️ Important:
  • Document ID must be exactly: 6UEfM5O4WNgUBWKSw38hsdVtaah1
  • The role field must be exactly: admin (lowercase, no spaces)
  • Don't click "Auto-ID" - paste the UID manually

Step 4: Save & Test

  1. Click "Save" in Firebase Console
  2. Clear your browser cache (Ctrl+Shift+Delete)
  3. Go to your login page: http://localhost:3000/login
  4. Login with your email and password
✅ Success!
After completing these steps, you should be able to access the admin dashboard without any "Access denied" errors.
